target recruiting myRetail


Deployment:

MongoDB
1. Install MongoDB and add to PATH
2. Start mongod with data path:
    example:'mongod.exe --dbpath=data'

Server
1. Clone git repo
2. install node modules:
    'npm install'
3. Check and modify config file as needed
4. Start server:
    'npm start'
